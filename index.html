Tags de títulos: <h1> até <h6>

As tags h1 até h6 em HTML são usadas para criar cabeçalhos de diferentes níveis em uma página web. O h1 representa o título principal, enquanto h2 a são usados para títulos de níveis subordinados, em ordem decrescente de importância. Essas tags são essenciais para estruturar e hierarquizar o conteúdo de uma página, facilitando a compreensão semântica do texto e proporcionando uma formatação visual distintiva.

Tag de parágrafo: <p>

A tag p em HTML é usada para definir parágrafos. Ela representa um bloco de texto que deve ser apresentado como um parágrafo, separado verticalmente do conteúdo adjacente por espaçamento. Essa tag é fundamental para estruturar o texto em uma página da web, dividindo-o em unidades lógicas e facilitando a leitura e compreensão do conteúdo. O uso apropriado da tag

contribui para a semântica da página e ajuda na formatação visual do texto, tornando a leitura mais organizada e legível.

Tag de hyperlink: <a>

(voltar)

A tag a em HTML é utilizada para criar hiperlinks, permitindo a criação de ligações entre páginas da web, documentos ou recursos online. Essa tag é essencial para a construção da estrutura de navegação em uma página, possibilitando que os usuários cliquem no link para acessar o conteúdo vinculado.
Dentro da tag a, é comum incluir o atributo href, que especifica o destino do link, indicando o endereço URL para o qual o usuário será direcionado ao clicar no link. Além disso, é possível incluir outros atributos, como target para definir como a página será aberta (por exemplo, em uma nova janela ou na mesma janela).
A tag a também é utilizada para criar links para âncoras dentro da mesma página, permitindo a navegação rápida para diferentes seções do conteúdo. A semântica dessa tag contribui para a organização e acessibilidade do conteúdo em uma página da web.

Tag de linha: <hr>

(voltar)

A tag hr em HTML é utilizada para criar uma linha horizontal, representando uma divisão ou separação visual entre elementos no conteúdo da página. Essa tag é comumente usada para dividir seções distintas ou para indicar uma mudança temática no conteúdo. O elemento hr não possui conteúdo e, portanto, é uma tag vazia.
Ao ser renderizada em um navegador, a tag hr geralmente resulta em uma linha horizontal que se estende pela largura do contêiner pai. A aparência visual exata pode ser estilizada usando propriedades CSS, caso seja necessário ajustar cor, espessura ou outros atributos visuais da linha horizontal.
Em resumo, a tag hr é utilizada para inserir uma linha horizontal que ajuda na organização visual do conteúdo, proporcionando uma separação clara entre diferentes seções ou elementos em uma página web.

Tags de índices: <sup> e <sub>

(voltar)

As tags sup e sub em HTML são usadas para controlar a apresentação de texto sobrescrito e subscrito, respectivamente, em relação à linha de base normal do texto.
sup: A tag sup é utilizada para criar texto sobrescrito. O texto dentro desta tag é exibido acima da linha de base regular do texto. Isso é frequentemente utilizado para representar expoentes em fórmulas matemáticas ou para indicar notas de rodapé.
sub: A tag sub é utilizada para criar texto subscrito. O texto dentro desta tag é exibido abaixo da linha de base regular do texto. Isso é comumente usado para representar índices em fórmulas químicas ou em situações onde o texto precisa ser exibido abaixo do nível padrão.
Ambas as tags são úteis para situações em que partes específicas do texto precisam ser elevadas ou rebaixadas visualmente em relação ao restante do conteúdo. Elas são especialmente comuns em contextos científicos, matemáticos e técnicos, onde é necessário representar informações de maneira precisa e legível.

Tag de citação: <blockquote> e </blockquote>

(voltar)

A tag blockquote em HTML é utilizada para representar um bloco de texto que é uma citação de outro recurso, como um livro, artigo ou site. Essa tag é empregada para destacar visualmente as citações, indicando claramente que o texto é uma reprodução de outra fonte.
Ao utilizar blockquote, os navegadores geralmente aplicam um recuo ou algum estilo visual específico para diferenciar o bloco de citação do restante do conteúdo. Além disso, é comum incluir informações adicionais sobre a citação, como o autor e a fonte, dentro do bloco de citação ou em elementos associados, como cite.
A tag blockquote é valiosa para fornecer contexto e atribuição a citações dentro de um documento HTML, contribuindo para a clareza semântica do conteúdo e melhorando a experiência do leitor ao entender a origem das informações citadas.
